## 遊戲操作說明
滑鼠控制飛船移動，按住發射子彈，鍵盤a和z控制左右旋轉。

## 特色介紹
- **靈活的操縱性**:因為是用滑鼠操作。
- **多種敵人**:有三個所以多。
- **獨特的美術風格**:玩家操控的飛船和子彈美術是原創的。

## 執行方式

1. 匯出為 `.jar` 檔
2. 雙擊執行
